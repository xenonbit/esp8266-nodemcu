Test your ESP8266.<br>

### Tests

- Built-in LED Turn on / Turn off
- WiFi Connection
- ROM Read / Write

### Commands
<hr/>

```
test
```

Starts the test process.

<hr/>

```
reset
```

Overwrites / Clears the ROM
